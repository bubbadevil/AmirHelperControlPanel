# Amir's Helper Control Panel

## Download

Download the latest version here:

https://github.com/bubbadevil/AmirHelperControlPanel/releases/latest

## Installation

1. Download the latest release ZIP.
2. Extract the ZIP to any folder.
3. Run `AmirHelperControlPanel.exe`.
4. Complete the First-Time Setup Wizard.
5. Log in using your Control Panel credentials.

## Features

* Discord Embed Builder
* Global Ban / Unban
* Role Sync Management
* User Management
* Permission System (Owner/Admin/Restricted)
* GitHub Auto Updates
* Control Panel Logging
